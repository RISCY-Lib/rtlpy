Required Tools
------------------

Python (>3.7)
~~~~~~~~~~~~~
Python version used

Sphinx
~~~~~~
Automatic document generation tool.

.. code-block:: bash

  pip install sphinx

sphinx-rtd-theme
~~~~~~~~~~~~~~~~
Used for HTML documentation theme with Sphinx

.. code-block:: bash

  pip install sphinx-rtd-theme

sphinx-markdown-builder
~~~~~~~~~~~~~~~~~~~~~~~~
Used to create the markdown documentation with Sphinx

.. code-block:: bash

  pip install sphinx-markdown-builder
